# ThinkGames — текущий статус проекта

## Что уже реализовано

### Главная страница

- карточки разделов `Математика`, `Нідерландська`, `Українська`
- верхний блок со звездой между аватаром и колокольчиком
- рабочий блок `Щоденний виклик` без кнопки `Продовжити`
- колокольчик с ежедневным сообщением-головоломкой
- красная точка у колокольчика исчезает после просмотра сообщения

### Daily Challenge

- состояние хранится в Supabase в `daily_challenge_progress`
- день засчитывается за прохождение хотя бы 1 игры
- повторные игры в тот же день не увеличивают серию
- за каждые 5 дней подряд начисляется 1 звезда
- обновление встроено в общий `saveGameResult(...)`

### Leaderboard

- отдельный маршрут `/leaderboard`
- нижнее меню обновлено: `Прогрес` заменён на `Leaderboard`
- в рейтинге показываются:
  - аватар
  - имя
  - общий XP
  - общий уровень
  - прогресс внутри уровня
  - количество звёзд
- источник данных: Supabase view `leaderboard_stats`

### Навигация

- нижнее меню: `Головна / Ігри / Leaderboard / Профіль`
- пункт `Ігри` ведёт на `/games`
- `/games` показывает все игры на одной странице, без деления на три блока

### Система XP

- за завершённую игру начисляется `10 XP`
- единая точка сохранения: `src/lib/points.ts`
- сохранение стало надёжнее: если записи категории нет, создаётся `insert`, иначе идёт `update`
- исправлено начисление XP в:
  - `Математика -> З'єднай пару`
  - `Нідерландська -> З'єднай слова`

### Озвучка

- подключён общий helper `src/lib/speech.ts`
- в озвучке исключён русский язык
- для `Math Connect Pairs` и `Dutch Connect Pairs`:
  - левая колонка — нидерландская озвучка
  - правая колонка — украинская озвучка
- для `Українська -> Напиши слово` включена украинская озвучка слова

### Сообщения дня

- создан отдельный список из 20 сообщений-головоломок
- каждый день показывается одно сообщение по порядку
- статус прочтения работает и для пользователя, и для гостя

### Ваги рівноваги

- обновлён внешний вид весов: тёмные разноцветные части, чаши одного цвета
- добавлено больше места сверху для гирь
- левая гиря визуально подогнана под чашу
- увеличена читаемость чисел на гирях
- устранён hydration mismatch за счёт клиентской генерации раунда
- внизу не показывается гиря с тем же числом, что и левая

### Орбітальна арифметика

- добавлены звуки через Web Audio API:
  - тяга
  - правильный астероид
  - проигрыш
  - победа
  - столкновение со стеной
- добавлена кнопка `звук вкл/выкл` справа от джойстика
- джойстик оставлен по центру
- поле адаптировано под телефон
- игра пересоздаёт уровень при изменении реального размера `canvas`
- скорректированы стартовые позиции ракеты и астероидов для компактного поля

## Текущая структура данных

- `profiles`
- `game_progress`
- `daily_challenge_progress`
- `leaderboard_stats`

## Что уже не актуально

- daily challenge больше не хранится только в `localStorage`
- пункт нижнего меню `Прогрес` больше не используется
- экран игр больше не ограничен разделами, есть единый `/games`
- блок `Щоденний виклик` на главной больше не заглушка

## Что важно помнить при разработке дальше

- новые игры должны сохранять прогресс через `saveGameResult(...)`
- leaderboard не должен собираться вручную на клиенте, источник — `leaderboard_stats`
- в озвучке нельзя использовать русский fallback
- для мобильных игр с физикой нужно учитывать реальный размер поля, а не только desktop-значения

## Следующие возможные направления

- топ-3 leaderboard на главной странице
- история ежедневных сообщений
- отдельные достижения помимо звёзд daily challenge
- дополнительные игры для раздела `Українська`
- усиление антидублей и антиабуза в XP-слое Supabase

## Базовая проверка

- `npx tsc --noEmit` должен проходить после изменений
