# ThinkGames — текущий статус проекта

## Реализовано

### Главная страница

- Есть карточки предметов: математика, нидерландский, украинский
- Показывается общий XP и прогресс по разделам
- Между аватаром и колокольчиком отображается счетчик звезд
- Блок "Щоденний виклик" больше не является заглушкой
- В блоке daily challenge убрана кнопка "Продовжити"
- Daily challenge показывает реальный прогресс серии по схеме `x/5`

### Щоденний виклик

- Один день засчитывается, если пользователь прошел хотя бы 1 игру за день
- Повторные игры в тот же день не увеличивают серию повторно
- За 5 дней подряд начисляется 1 звезда
- Состояние хранится в Supabase в таблице `daily_challenge_progress`
- Обновление daily challenge встроено в общий `saveGameResult(...)`

### Leaderboard

- Реализован отдельный экран `/leaderboard`
- В нижнем меню пункт "Прогрес" заменен на `Leaderboard`
- В leaderboard отображаются:
  - аватар
  - имя
  - общий XP
  - общий уровень
  - прогресс внутри текущего уровня
  - количество звезд
- Сортировка идет по звездам, затем по общему XP
- Данные читаются из Supabase view `leaderboard_stats`

### Профиль и авторизация

- Регистрация и вход через Supabase Auth
- Профиль пользователя хранится в `profiles`
- Аватар и имя используются на главной и в leaderboard

### Игры

- Математика:
  - "Знайди пару"
  - "З'єднай пару"
  - "Ваги рівноваги"
  - "Кульковий сапер"
- Нидерландский:
  - "З'єднай слова"
  - DutchWordle
- Украинский:
  - "Напиши слово"

### Система очков

- За завершенную игру начисляется фиксированно 10 XP
- XP хранится в `game_progress`
- Категории прогресса:
  - `math`
  - `ukrainian`
  - `dutch`

### UI и UX

- Используется единый `GameEndModal`
- Нижнее меню унифицировано
- Интерфейс mobile-first
- Применяются анимации Framer Motion

## SQL / Supabase

### Уже используются

- `database-setup.sql` — profiles и триггер профиля
- `daily-challenge-setup.sql` — daily challenge table + RLS
- `leaderboard-setup.sql` — view `leaderboard_stats`

### Текущая структура данных

- `profiles`
- `game_progress`
- `daily_challenge_progress`
- `leaderboard_stats` (view)

## Что уже не актуально

- Daily challenge больше не хранится в `localStorage`
- Пункт нижнего меню "Прогрес" удален
- Leaderboard больше не находится "в планах", он уже реализован

## Ключевые маршруты
